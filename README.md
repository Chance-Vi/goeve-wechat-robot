## goeve-wechat-robot

_这是一个简单的智能助理微信机器人,主要有以下功能:_

1. 自动通过好友申请，主动发送问候语（自定义）
1. 回复关键字自动拉入微信群（自定义）
1. 群内新人加入，主动发送欢迎语（自定义）

查看原理、配置及源码使用说明,请前往知乎live:
